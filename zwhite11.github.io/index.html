Penguin Basketball Stat Site 2018
Project Repository

Site containing the statistics for the Penguin Basketball Association during the 2018 basketball season. Includes Men’s and Women’s stats as well as team and individual stats. Team and individual stats can be filtered to those against specific opponents, home game, away games, wins, and losses.

Penguin Basketball Stat Site 2018 (Mongoose) (Repository)

This project is the same as the Penguin Stat Site except that the data is stored with MongoDB using Mongoose and Node.js. The data is no longer stored locally in .json files.

Pizzapolis Mock Pizza Ordering Application (Repository)

Pizzapolis is a mock pizza ordering application created as a final group project for the course “Web Application Development with Java” offered at Johns Hopkins University (605.782). It was developed in Netbeans IDE using HTML, JSP, JavaScript, and Java Servlets for it’s front-end UI. Pizzapolis uses JavaScript for short-term client-side persistence. It also uses MySQL and Hibernate ORM for it’s long-term server-side persistence layer. The application also sends order confirmation email using JavaMail API and Javax. JSON and invisible form entries are used to pass data from client to server.
